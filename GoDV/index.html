开始安装

你准备好了运行环境了吗？查看我们的安装说明，您将获得提示

$ git clone https://github.com/xuthus5/GoDV.git
$ cd GoDV/Server    //启动服务端
$ go run .
$ cd GoDV/Client    //启动客户端
$ npm install
$ npm run serve
